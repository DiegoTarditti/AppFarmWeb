Cómo se calcula el pedido

Promedio ponderado: Se calcula sobre los 11 meses completos de historial, dando más peso a los meses recientes. Si el mes actual tiene datos parciales, se prorratea para estimar el mes completo.

Tendencia (slope): Regresión lineal sobre los meses de datos. Indica si las ventas suben (↑), bajan (↓) o se mantienen (→). Se usa para proyectar la demanda futura.

Estacionalidad: Cada mes tiene un índice estacional (ventas del mes / promedio). Los meses de pico se marcan como "Pico" y los de baja como "Baja".

Proyección: Para cada día del período de pedido se estima: promedio × estacionalidad + tendencia. Se suman los días y se resta el stock actual.

Producto CRÓNICO: Se detecta automáticamente cuando las ventas son muy estables mes a mes (coeficiente de variación < 30% y al menos 8 de 11 meses con ventas). Para estos productos la tendencia se amortigua al 20%, priorizando el promedio histórico ya que la demanda es constante (pacientes crónicos).

Rotación: A Alta (≥ {{ rot_alta_min }} u/mes) · M Media (≥ {{ rot_media_min }} u/mes) · B Baja

Gráfico: Barras azules = ventas históricas. Barras verdes = proyección. Línea naranja = tendencia. Línea amarilla punteada = promedio ponderado. Línea roja punteada = stock actual. El último mes (con *) muestra la venta parcial + extensión ámbar del prorrateo estimado.

Cobertura: Días que el stock actual alcanza a cubrir según el promedio de ventas diario.
