# Pantalla: Catálogo ObServer

> ⚠ STATUS: PENDIENTE

**Ruta**: `/obs/productos`

## Para qué sirve

Ver el catálogo completo de ObServer (~122k productos) con stock actual, ventas históricas y datos de laboratorio.

## Filtros

- Búsqueda por nombre o código alfabeta.
- Filtro por laboratorio.

## Columnas

- Descripción
- Laboratorio (ObServer)
- **Monodroga** — clickeable → te lleva a [Estadísticas por droga](./estadisticas_drogas.md) filtrado.
- Stock actual
- Promedio mensual 3m / 12m
- Código alfabeta
- EAN local (si está vinculado vía bridge `productos.observer_id`)
